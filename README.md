Este proyecto llamado 'Willichedunguay' consiste en un videojuego 2D, de tipo quiz, que tiene como propósito enseñar el idioma williche, una variante del mapudungun. El juego contará con 3 niveles: 'Principiante', 'Intermedio y 'Avanzado', y cada nivel contarpá con igualmente 3 subniveles. Actualmente se encuentra en desarrollo.

Archivos y carpetas en el repositorio:
-main.py : Inicializa el juego.
-game.py : Contiene la lógica y el bucle del juego.
-settings.py :Contiene las variantes globales del juego, actualmente solo contiene el tamaño de la ventana y colores.
-states : Esta carperta contiene los archivos que manejan los estados del juego.
  -level_selector.py : Contiene lo que es la pantalla para seleccionar los niveles del juego.
  -menu.py : Contiene lo que es la pantalla del menú para comenzar el juego.
-assets : Esta carpeta contiene archivos estáticos.
  -images : Contiene las imágenes.
  -font : Contiene la tipografía del juego.
